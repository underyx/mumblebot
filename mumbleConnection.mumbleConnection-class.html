Class mumbleConnection

This class represents a persistent connection to a mumble Server via TCP. Id doesn't support UDP and taking in account the fact that UDP is optional it most likely never will.

__init__(self, host, password, port, nickname, channel)
(Constructor)

 

Creates a mumble Connection but doesn't open it yet.

Parameters:
  • host (String) - Mumble server to connect to, as hostname or IP address
  • password (String) - Server password, if the server doesn't have one, leave it empty or put in whatever you like to.
  • port (String) - Port on which the mumble server listens.
  • channel (String) - Channel name the bot should join.

addChatCallback(self, trigger, function)

 

Adds a function and a trigger for that function. Will execute the given function if the "Trigger" String occurs in channel tex".

Parameters:
  • trigger (String) - Text trigger, currently NO regexp support
  • function (Python Function) - Function to be called, Strings it returns are written back to the channel.
